# Python Syntax

## 縮排

python使用縮排來表示一個程式區塊，有別其他語言使用花括號等符號。

一般根據PEP8，每層縮排使用4個空格，不過這只是一種程式撰寫風格，使用其他數量的空格來縮排並不會出錯。

```py
if 5 > 2:
    print("Five is greater than two!")
```

## 註解

python使用`#`當作註解宣告符號

```py
print("Hello") # 輸出Hello
```

## 變數宣告

python沒有變數宣告的關鍵字，直接以變數名稱指定值即可使用

```py
x = 5
print(x) # output: 5
```
